Help

Get Pokemon by type

/type/pokemon_type/best

pokemon_type (Required) - return only pokemon of the given type

best (Optional, default = false) - return only the best pokemon of the given type

Get stats of one Pokemon

/pokemon/pokemon_name

pokemon_name (Required) - return the stats of the given pokemon

Get Pokemon from a given generation

/generation/generation_number/legendary

generation_number (Required) - returns the pokemon from the given genration number

legendary (Optional, default = false) - returns only the legendary pokemon from the given generation
